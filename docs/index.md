# SharpPgQuery

SharpPgQuery provides C# bindings for the PostgreSQL `libpg_query` parser so you can inspect SQL syntax trees and normalization results from managed code.

- Parse SQL text into a Roslyn-like syntax tree
- Access diagnostics produced by the PostgreSQL parser
- Normalize SQL statements with placeholder parameters
